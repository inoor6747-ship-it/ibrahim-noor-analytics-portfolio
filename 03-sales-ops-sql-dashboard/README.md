# Sales Operations Analytics

## Purpose
Business teams track revenue but often lack clear performance drivers.  
This project builds a SQL reporting layer and executive dashboard to analyze sales behavior.

## What This Project Shows
- SQL KPI modeling
- Business metric design
- Performance analysis

## Data Pipeline
- Raw order and return data stored in SQLite
- SQL transforms generate reporting tables
- KPI tables exported for BI reporting

## Metrics Analyzed
- Revenue
- Orders
- Average Order Value
- Margin %
- Return rate by channel
- Revenue by region
- Top products
